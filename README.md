# Desktop.WinForm.SRI.Download.XML

Aplicación de escritorio en Windows Forms (C#) para descargar y visualizar archivos XML del Servicio de Rentas Internas (SRI) de Ecuador.

## Descripción

Esta aplicación permite a los usuarios conectarse a los servicios del SRI y descargar archivos XML relacionados con información fiscal. Cuenta con una interfaz gráfica intuitiva que facilita la navegación y visualización de los datos descargados.

## Características

- Interfaz gráfica simple y amigable (WinForms).  
- Descarga de archivos XML directamente desde los servicios web del SRI.  
- Visualización de los archivos XML dentro de la aplicación.  
- Configuración de parámetros a través del archivo `App.config`.  

## Requisitos

- Windows 10 o superior.  
- .NET Framework 4.7.2 o superior.  
- Conexión a internet activa para acceder a los servicios del SRI.
